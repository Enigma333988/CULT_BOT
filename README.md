# CULT_BOT 🤖

Консольный Python-бот для отправки сообщений в Telegram через Bot API.

---

## 🚀 Возможности

* 📤 Отправка сообщений в Telegram
* 🔐 Безопасное хранение токена через `.env`
* 🌐 Поддержка прокси
* ⚡ Простая настройка и запуск

---

## 📦 Структура проекта

```
CULT_BOT/
├── bot.py
├── requirements.txt
├── .env.example
├── .gitignore
└── README.md
```

---

## ⚙️ Установка

### 1. Клонировать репозиторий

```
git clone https://github.com/Enigma333988/CULT_BOT.git
cd CULT_BOT
```

### 2. Установить зависимости

```
pip install -r requirements.txt
```

---

## 🔑 Настройка

Создай файл `.env` на основе шаблона:

```
cp .env.example .env
```

Заполни:

```
TOKEN=your_bot_token_here
CHAT_ID=-1001234567890
PROXY=
```

---

## 🧠 Переменные окружения

| Переменная | Описание             |
| ---------- | -------------------- |
| `TOKEN`    | Токен Telegram-бота  |
| `CHAT_ID`  | ID чата или канала   |
| `PROXY`    | Прокси (опционально) |

---

## ▶️ Запуск

```
python bot.py
```

После запуска вводи сообщения в консоль — они будут отправляться в Telegram.

---

## ❌ Выход

Введи:

```
exit
```

или

```
q
```

---

## 🔐 Безопасность

* Никогда не загружай `.env` в GitHub
* Не публикуй `TOKEN`
* Используй `.env.example` как шаблон

---

## ⚙️ Зависимости

* requests
* python-dotenv

---

## 🧠 Как это работает

1. Программа читает `.env`
2. Берёт `TOKEN` и `CHAT_ID`
3. Отправляет HTTP-запрос в Telegram API
4. Сообщение появляется в чате

---

## 🌐 Telegram API

Используется официальный метод:

```
https://api.telegram.org/bot<TOKEN>/sendMessage
```

---

## 📌 Примечания

* Если бот не отправляет сообщения — проверь `TOKEN`
* Если ошибка сети — проверь `PROXY`
* Убедись, что бот добавлен в чат
