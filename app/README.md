Структура пакета:

- `app/legacy.py` — текущая рабочая реализация бота, перенесённая из старого `bot.py`.
- `app/config.py` — конфиг и runtime-константы.
- `app/state.py` — runtime-состояние, JSON-хранилище и UI state.
- `app/domain.py` — бизнес-логика заказов, каталога, архива и отчётов.
- `app/platforms.py` — Telegram/MAX API и платформенные helper-функции.
- `app/ui.py` — тексты, клавиатуры и отправка экранов.
- `app/handlers.py` — команды, callback и update-handlers.
- `app/polling.py` — polling-функции.
- `app/main.py` — точка входа пакета.

Безопасный первый шаг сделан через фасадные модули: поведение не меняется, а дальнейшие правки можно переносить из `legacy.py` в профильные файлы постепенно, не ломая запуск.
