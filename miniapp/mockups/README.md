# Mockups

Складывай сюда макеты для Mini App.

Текущий рабочий формат:

- размер: `1080x1920`
- ориентация: portrait
- базовое имя для автопоказа: `current.png`
- также поддерживаются: `current.jpg`, `current.jpeg`, `current.webp`

Если нужно открыть конкретный файл, можно использовать URL:

- `/?mockup=home-1080x1920.png`

Эта папка используется режимом mockup preview в `miniapp/index.html`.
